# 기능목록

## 입력 - InputView
-[ ] 자동차 이름(출전자 명단)을 입력한다.
  -[ ] 하나의 문자열로 쉼표(,)로 각 이름을 구분한다.
  -[ ] (e) 이름이 5글자를 초과할시 예외 발생
-[ ] 몇 번 이동할건지(목적지까지의 거리)를 입력한다.
  -[ ] 숫자 형태로 입력받는다.
  -[ ] (e) 입력에 숫자 외에 형태가 있을 경우 예외 발생

## 자동차 - Car
-[ ] 출전자 이름을 기억한다.
-[ ] 현재 위치를 기억한다.
-[ ] 이동을 진행한다.
  -[ ] 전달받은 값이 4이상일 경우 한칸 전진한다.
  -[ ] (e) 전달받은 값의 범위가 0~9 밖인 경우 예외 발생

## 레이싱 - Game
-[ ] 참가자 명단을 받는다.
-[ ] 레이싱을 진행한다.
  -[ ] 각 자동차가 전진한다.
  -[ ] 우승자가 나왔는지 확인한다.
-[ ] 우승자를 발표한다.

## 출력 - OutputView
-[ ] 자동차 이름 입력을 안내한다.
-[ ] 시도할 횟수 입력을 안내한다.
-[ ] 레이싱 실행 결과를 출력한다.
-[ ] 우승자 정보를 출력한다.
-[ ] 예외를 출력한다.
