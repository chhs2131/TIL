# 넘파이
- 넘파이는 대용량 배열과 행렬 연산을 빠르게 수행함
- 파이썬의 기본 리스트에 비하여 처리속도가 매우 빠름
- 넘파이는 ndarray를 제공하며 이는 C언어에 기반한 배열 구조임
